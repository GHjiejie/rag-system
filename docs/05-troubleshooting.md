# 05｜常见问题排查手册

本篇用于回答卡顿、网络异常、应用闪退、启动问题等。

## 总体排查原则

1. 先复现，再修改。
2. 一次只改一个变量。
3. 先做低风险操作（重启、断开外设、关闭后台）。
4. 记录“现象-操作-结果”。

## 场景一：系统卡顿

### 快速步骤

1. 打开活动监视器看 CPU/内存占用。
2. 关闭异常高占用应用。
3. 检查磁盘剩余空间是否过低。
4. 重启后只开必需应用复测。

### 常见原因

- 浏览器标签过多。
- 后台同步/下载持续运行。
- 内存不足导致频繁交换。

## 场景二：Wi-Fi 不稳定

### 快速步骤

1. 先排除路由器问题（其他设备是否同样异常）。
2. 重新连接网络并重启路由器。
3. 关闭 VPN/代理后测试。
4. 更换 DNS 测试连通性。

## 场景三：应用闪退/打不开

### 快速步骤

1. 更新应用到最新版本。
2. 检查权限（磁盘、网络、麦克风等）。
3. 清理应用缓存并重启。
4. 重新安装应用。

## 场景四：启动异常

### 快速步骤

1. 断开外接设备后重启。
2. 进入安全模式验证。
3. 检查启动项是否异常。

## 外接显示器无画面

1. 确认线材支持目标分辨率与刷新率。
2. 换接口/换线排除硬件问题。
3. 在显示设置中手动检测和调整参数。
4. 绕过扩展坞直连测试。

## 何时需要专业支持

- 无法开机或频繁内核崩溃。
- 硬件异常噪音、明显发热异味。
- 重要数据存在丢失风险。

建议优先备份后再进行深度维修。
