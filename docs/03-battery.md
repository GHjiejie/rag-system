# 03｜电池、续航与充电习惯

本篇用于回答“续航差怎么办”“电池健康下降快是否正常”等问题。

## 先看结论

- 电池属于消耗品，健康度随时间下降是正常现象。
- 高温、长期高负载、频繁极端充放电会加速老化。
- 日常插电办公并非一定伤电池，关键是温度与负载管理。

## 影响续航的常见因素

- 屏幕亮度过高。
- 浏览器标签页和后台应用过多。
- 云同步、视频会议、下载任务持续运行。
- 外接高分辨率显示器导致功耗上升。
- 系统或应用异常占用 CPU。

## 充电习惯建议

1. 避免长时间高温环境使用与充电。
2. 不必强迫每次充到 100% 或放到 0%。
3. 长期插电办公时，关注机身温度和散热条件。
4. 使用质量可靠的充电器与线材。

## 电池健康排查流程

1. 查看循环次数与健康状态。
2. 打开活动监视器，检查高能耗进程。
3. 观察是否某应用异常常驻。
4. 测试“只开基础应用”时的续航基线。
5. 必要时重启或进入安全模式验证。

## 发热与风扇（如适用）

- 高负载发热属于正常现象。
- 空闲高温通常不正常，优先排查后台任务。
- 长期高温会影响体验和电池寿命，应及时处理。

## FAQ

### Q1：待机掉电快正常吗？
轻微掉电正常；若异常快，优先排查网络、同步和后台任务。

### Q2：边充边用会伤电池吗？
正常使用不会显著伤害，关键是避免高温和劣质配件。

### Q3：什么时候考虑更换电池？
当健康度明显下降并影响工作时长，建议评估更换。
