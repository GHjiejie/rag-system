# 06｜配件、扩展坞与外接显示器

本篇用于回答“买什么配件更实用”“显示器和扩展坞如何选”等问题。

## 配件优先级

1. 原装或可靠品牌充电器与线材。
2. 外接显示器（按工作场景）。
3. 扩展坞（接口不足时）。
4. 外置 SSD（大文件与项目归档）。

## 充电器与线材建议

- 选择支持所需功率的充电器。
- 线材需支持对应功率和数据规格。
- 避免长期使用质量不明的低价配件。

## 扩展坞选择要点

- 先列出你真实需要的接口（HDMI、USB-A、USB-C、网口、读卡器）。
- 关注显示输出能力（分辨率、刷新率、显示器数量）。
- 关注供电能力（PD 回充功率）。
- 优先选择口碑稳定、散热良好的型号。

## 显示器连接建议

- 办公建议优先高分辨率与清晰度。
- 设计类工作关注色域与色准。
- 高刷新场景确认链路全程支持（线材+扩展坞+显示器）。

## 常见问题

### Q1：为什么接上显示器只有 30Hz？
通常是线材或扩展坞规格不足，或显示设置未正确配置。

### Q2：扩展坞发热正常吗？
轻度发热正常；若异常烫且不稳定，建议更换更高规格设备。

### Q3：外置 SSD 速度不达标怎么办？
检查接口协议、线材规格、硬盘盒芯片与文件系统格式。

## 移动办公建议清单

- 轻量氮化镓充电器
- 一条高质量全功能线
- 小型扩展坞
- 便携 SSD
- 电脑支架与收纳包
