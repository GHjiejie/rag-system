# MacBook 知识库总览

本知识库用于给 MacBook 相关问答、检索和 RAG 测试提供基础内容。文档以中文编写，适合回答入门问题、选购问题、使用问题和排障问题。

## 覆盖主题

- MacBook 产品线区别
- Apple 芯片与配置选择
- 电池续航与充电习惯
- macOS 日常使用方法
- 常见问题排查
- 配件、扩展坞和外接显示器

## 适合回答的问题示例

- MacBook Air 和 MacBook Pro 有什么区别？
- M 系列芯片怎么选？
- 16GB 内存够不够？

- 学生、开发者、办公用户分别适合买哪类 MacBook？

如果你要把这些文档用于知识库检索，建议优先围绕以下形式提问：

- 直接问问题，例如“MacBook 为什么待机掉电快？”
- 带上场景，例如“MacBook 连接会议室投影仪没有画面怎么办？”
- 带上目标，例如“我想买一台适合写代码和轻度剪辑的 MacBook”

这样的提问方式更容易命中对应文档块。
