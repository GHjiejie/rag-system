# 04｜macOS 日常使用与效率

本篇用于回答“新手怎么快速上手 macOS”“如何提升效率”等问题。

## 基础设置（建议首日完成）

- 登录 Apple 账号并开启必要同步。
- 配置输入法、触控板手势、Dock 与控制中心。
- 打开系统更新，保持安全补丁及时。
- 设置 Time Machine（或其他备份方案）。

## 常用效率功能

- Spotlight：全局搜索应用、文件、设置。
- 快捷键：复制粘贴、切换窗口、截图、锁屏。
- 分屏与桌面空间：多任务时提高效率。
- 聚焦模式：减少通知干扰。

## 文件管理建议

- 统一项目目录命名规范（如按日期/项目分类）。
- 定期清理下载目录与临时文件。
- 大文件建议外置硬盘或云端归档。

## 开发者常用建议

- 安装包管理工具并管理依赖版本。
- 使用虚拟环境隔离 Python 项目。
- 终端中保留常用别名与脚本。
- 遇到环境问题先确认当前 shell 与路径。

## 安全与隐私

- 按需授予应用权限（麦克风、屏幕录制、文件访问）。
- 不安装来源不明的软件。
- 定期检查启动项与登录项。

## FAQ

### Q1：为什么应用提示“已损坏/无法打开”？
常见于来源不可信或签名问题，建议从官方来源重新安装。

### Q2：系统更新后变慢怎么办？
先观察索引和后台任务是否仍在运行，再评估磁盘空间与内存压力。

### Q3：如何减少干扰、提高专注？
使用聚焦模式 + 分屏 + 通知分级策略。
